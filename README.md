# Tolmacheva_KPO_CW1

## Описание проекта
Это консольное приложение для учета финансов, реализующее:
- Управление счетами, категориями и операциями
- Подсчет баланса и аналитика доходов/расходов
- Импорт и экспорт данных в JSON и CSV
- Использование паттернов проектирования (`Facade`, `Factory`, `Visitor`, `Template Method`)
- Обработка ошибок и валидация пользовательского ввода

## Установка и запуск

### Установить необходимые зависимости
Проект требует **.NET 7.0**. Установите его, если он отсутствует:

## Нужно клонировать проект или скачать файлы
git clone https://github.com/katetolmacheva/Tolmacheva_KPO_CW1
cd Tolmacheva_KPO_CW1

## Собрать и запустить программу
dotnet build
dotnet run

## Проект включает юнит-тесты (XUnit). Чтобы запустить тесты, выполните:
dotnet test
